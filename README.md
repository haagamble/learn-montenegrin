# Montenegrin Phrase Trainer

A small static React app to help learn basic Montenegrin phrases and flashcards.

## Start locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the app:
   ```bash
   npm run dev
   ```
3. Open the local URL shown by Vite.

## Build for production

```bash
npm run build
```

## GitHub Pages

You can publish the built `dist` folder to GitHub Pages. A common setup is to use a `gh-pages` branch or GitHub Pages deployment from the `main` branch's `docs/` folder.
